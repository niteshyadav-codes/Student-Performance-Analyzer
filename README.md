# Student-Performer-Analyzer
A beginner-friendly data analysis project built with Python and Pandas to explore, analyze, and generate insights from student performance data.
## Features

* Read CSV file
* Display dataset information
* Calculate average marks
* Find highest and lowest marks
* Identify topper(s)

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Dataset

* 100 Student Records
* Name
* Age
* Course
* Marks
* City

## Results

* Total Students: 100
* Average Marks: 79.48
* Highest Marks: 100
* Lowest Marks: 60
